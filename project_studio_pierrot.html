Technical Implementation

Architecture

The project follows a classic ELT (Extract, Load, Transform) pipeline:

  • Data Acquisition: Python scripts fetch data from the Jikan API (MyAnimeList) and generate simulated internal business metrics.
  • Data Warehouse: A Star Schema implemented in SQLite with dimension tables (Anime, Season) and fact tables (Metrics, Financials, Marketing).
  • Analysis: Complex SQL queries derive insights on retention, profitability, and quality trends.
  • Visualization: A custom web dashboard built with Chart.js and Tailwind CSS consumes JSON data exported from the warehouse.
