## ğŸ› Troubleshooting

### Build Errors
- Clear Vite cache: `Remove-Item -Recurse -Force node_modules/.vite`
- Reinstall dependencies: `npm clean-install`

### Dev Server Issues
- Check port availability (default: 5173)
- Try `npx kill-port 5173` if port is blocked

### Production Build 404s
- Ensure `base: '/Portfolio/'` is set in `vite.config.js`
- Verify all asset paths are correct
